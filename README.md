# Velvet Glow Website Wireframe

A low-fidelity wireframe designed for the “Velvet Glow” luxury beauty and skincare website.

## Project Overview
This project focuses on creating a clean and structured website wireframe to improve user experience and website layout planning before final UI design.

## Sections Included
- Navbar
- Hero Section
- Featured Collection
- About Section
- Benefits Section
- Testimonials
- Footer

## Features
- Modern wireframe layout
- Clean user flow
- Proper spacing and alignment
- Responsive structure planning
- Luxury beauty website aesthetic
